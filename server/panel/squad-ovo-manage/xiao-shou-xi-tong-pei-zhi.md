# 销售系统配置

销售系统是 Squad ovo Manage 的核心功能之一。服主可通过该系统售卖预留席位，以此获得运营收益，为服务器日常运维提供稳定资金支撑。

## 站点配置

### 信息配置

我们已精简站点基础配置流程，仅需在基础设置内填写三项核心内容，即可快速完成部署：

* 站点名称
* 联系客服链接
* 主题模板

站点名称您可以根据自己的喜好填写，例如：**幻想之境 - 赞助平台**

联系客服链接是需要填写 QQ 好友添加链接，<mark style="color:red;">**切勿直接填写 QQ 号码**</mark>。

主题模板提供多款模板自由选择，可根据视觉喜好自行切换。

### 支付方式设置

支付方式控制用户购买预留位的付款渠道

目前系统内置三种主流支付渠道：

* 支付宝支付
* 微信支付
* QQ 支付

你可根据运营需求，自主开启或关闭任意支付方式，灵活调配使用。

### 支付设置

需先前往【**支付接口申请地址**】完成商户注册，再将商户 ID、商户密钥填写至对应配置栏；

同时需在支付接口平台配置收款账号，全部配置完成后，站点才能正常收款结算。

<details>

<summary>支付接口申请教程</summary>

支付接口申请注册地址：[https://mzf.nansai.cc/User/Login/Reg](https://mzf.nansai.cc/User/Login/Reg)

1. 账号注册：填写用户名、密码、邮箱，完成验证码验证即可注册成功。
2. &#x20;登录后，点击左侧列表**会员套餐**，按需选择月 / 季 / 年套餐购买（用于支付接口通道使用费，月费约 8.25 元）。
3. 套餐购买完成，点击左侧**通道列表**，添加支付宝 / 微信收款通道，首次添加后点击**配置**按钮完成通道上线。
4. 通道上线后，进入左侧**我的设置 - API 对接**，将页面展示的 PID、KEY，分别填入商户 ID、商户密钥即可。

</details>

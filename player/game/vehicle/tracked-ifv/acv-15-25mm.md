# ACV-15 25mm

ACV-15 IFV 是土耳其 FNSS 公司开发的一款两栖步兵战车

## 基本数据

| 数据名称     | 值      |
| -------- | ------ |
| 载具血量     | 1250   |
| 最大载员人数   | 11     |
| 最大载弹量    | 600    |
| 是否为两栖载具  | 是      |
| 是否具备 STA | 是      |
| 瞄具可缩放倍数  | 1x、10x |

该问题通常是由于配置不正确或音频输入设备未正常连接至您的计算机所致。

## 我不知道该怎么配置麦克风

在游戏内点击 「设置」 - 「音频」 ，即可找到音频输入设备选项。在音频输入设备的下拉选择框中，选中您正在使用的麦克风；选择完成后，您可以对着麦克风说话测试 —— 此时选择框下方的音量条会随您的声音分贝实时波动，若出现波动则说明麦克风配置成功。

## 音频设置无法找到麦克风

请按以下步骤检查麦克风设置：

1. 打开系统设置，进入 「系统 - 声音」 页面，在 “输入” 选项中查看是否能找到目标麦克风；
2. 若未找到，可在当前页面下滑，进入 「高级 - 所有声音设备」 ，检查该麦克风是否被禁用——若已禁用，点击允许应用和 Windows 将此设备用于音频；
3. 若在“所有声音设备”中仍未发现目标麦克风，大概率是麦克风未正常连接电脑或驱动未正确安装，建议联系设备经销商协助排查。

## 插入了麦克风也选择了麦克风

若您已在游戏内正确配置音频输入设备，但按下无线电发言键时仍提示 “No Input Device”，且音量条未随声音分贝动态变化，大概率是 Windows Audio 服务异常导致。

#### 修复步骤

1. 在 Windows 搜索框中输入 「命令提示符」 ，并选择 「以管理员身份运行」 ；
2. 将以下命令复制粘贴至命令提示符窗口，执行后即可重启 Windows Audio 服务：

```
net stop Audiosrv && net start Audiosrv
```

若窗口提示 「Windows Audio 服务已经启动成功」 ，则说明修复完成，可重新测试无线电发言功能

## 前言

被 EAC（**Easy Anti - cheat**）封禁后，意味着您游玩 Squad 的 Steam 账号以及 EOS ID 被 EAC 列入了黑名单。在被解封前，您无法游玩受到 EAC 保护的服务器。这不仅会中断您的游戏体验，还可能给您带来诸多不便。因此，了解如何申诉解封是非常必要的。

## 准备

* **EOS ID 或 Steam ID**

这两个 ID 是申诉时用于确认您账号身份的重要信息。虽然系统一般能够自动获取，但为了确保申诉流程顺利，建议您提前了解并牢记自己的 EOS ID 和 Steam ID，以便在必要时手动准确填写。

* 您的 **电子邮箱**

务必提供一个常用且有效的电子邮箱。官方会通过此邮箱向您发送申诉相关的通知、反馈等重要信息，若邮箱无效或长期未查看，可能导致您错过关键消息，影响申诉进度和结果。

## 申诉

1. 策略查看：在决定申诉前，您首先要查看[管理帐户暂停的策略](https://www.easy.ac/en-US/support/game/account/bans)。仔细研读该策略，深入了解 EAC 关于封禁和解封的具体规定、判定标准等内容。这有助于您判断自身申诉的合理性，以及预估申诉成功的概率，从而更明智地决定是否进行申诉。
2. 申诉地址：前往[申诉地](https://www.easy.ac/en-US/support/contact/appeal?game=55)[址](https://www.easy.ac/en-US/support/contact/appeal?game=55)开始申诉流程。在该页面，您将按照提示逐步完成一系列申诉操作。

**申诉流程：**

* 填写信息：进入申诉页面后，请严格按照提示准确填写您的 EOS ID 或 Steam ID、电子邮箱等基本信息。此外，可能还需要您填写一些与封禁相关的具体细节，例如您被封禁的大致时间、是否清楚可能导致封禁的原因等。这些信息填写得越准确、详细，越有助于申诉审核人员了解您的情况。
* 申诉理由阐述：这是整个申诉过程中最为关键的部分。您需要用清晰、诚恳且有条理的语言阐述自己认为被误封的理由。如果您在游戏过程中有能够证明自己遵守规则的证据，如游戏内的正常操作截图、与队友的聊天记录等，务必在相应位置上传或详细说明。比如，若您在被封禁前仅仅是正常使用游戏内置的语音交流功能，并未使用任何外挂或作弊软件，那就详细描述当时的操作情况以及周围环境等相关细节。
* 提交申诉：在确认所有填写和上传的信息准确无误后，点击提交按钮完成申诉。提交成功后，系统会自动生成一个申诉单号，请务必妥善保存这个单号，后续您可凭借它在官方指定的渠道查询申诉进度。
